Site CMS

这里是站点内容后台入口。当前仓库已经接入 Pages CMS 配置，可直接编辑博客标题、正文、Front Matter 和图片，并提交回 GitHub 仓库。

首次使用时，先用 GitHub 账号登录 Pages CMS，然后选择仓库 sunlongyu/sunlongyu.github.io 和分支 main。

打开 Pages CMS 查看 CMS 配置
  1. 进入 Pages CMS 后，打开 Blog Posts。
  2. 现在博客内容目录已经统一为小写路径，Blog Posts 应该会直接列出文章。
  3. 编辑标题、正文或上传图片；保存后会直接提交到仓库，GitHub Pages 会自动重新发布。
